ASSETS FOLDER
=============

Please add the following image files to this folder:

1. icon.png (1024x1024) - App icon
2. splash.png (1284x2778) - Splash screen image  
3. adaptive-icon.png (1024x1024) - Android adaptive icon
4. favicon.png (48x48) - Web favicon

Recommended:
- Use a gold coin or spinning wheel theme
- Use dark background (#0f172a) for splash screen
- Make icons with transparent backgrounds for best results

You can generate these using:
- https://www.appicon.co/
- https://romannurik.github.io/AndroidAssetStudio/
- Or any image editor like Figma, Sketch, or Photoshop
